# Cloud SaaS Task & Project Manager

A cloud-native, AI-assisted task and project management SaaS application.

## Tech Stack

- Frontend: Next.js (TypeScript, Tailwind)
- Backend: Node.js, TypeScript, Express, Prisma
- Database: PostgreSQL
- Cloud: AWS (ECS/Fargate, RDS, S3, ALB)
- Infra as Code: Terraform
- CI/CD: GitHub Actions
- Extras: AI-assisted subtasks & prioritization

## Structure

- `backend/` — API, auth, business logic
- `frontend/` — Web UI
- `infra/` — Terraform & deployment config
